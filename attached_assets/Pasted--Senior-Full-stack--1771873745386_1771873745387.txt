Создание ИИ-конструктора сайтов «НейроЗодчий»
Роль: Ты — Senior Full-stack разработчик. Твоя задача — создать веб-приложение для генерации современных веб-сайтов на базе Gemini 3.1 Pro API.

Основная концепция:
Приложение на русском языке, позволяющее новичкам создавать сайты через текстовый запрос, шаблон или по скриншоту. Результат — чистый HTML/CSS/JS код, готовый к выгрузке.

Технический стек:

Frontend: Next.js или React (Tailwind CSS).

Анимации: Framer Motion (обязательно для плавных переходов и SVG-анимаций).

Дизайн: Ультрасовременный скевоморфизм (Skeuomorphism) с элементами Glassmorphism. Интерфейс должен выглядеть «дорого» и тактильно.

Backend: Node.js + Supabase (для авторизации и хранения истории проектов).

Структура приложения (Страницы):

Landing Page: Продающая страница с описанием преимуществ и блоком из 4 тарифов (Бронза, Серебро, Золото, Платина), привязанных к количеству токенов API.

Auth: Вход и регистрация (Email/Google).

Dashboard (Личный кабинет):

История созданных проектов (карточки с превью).

Кнопка «Создать проект», после чего стильный поп ап с картинками и текстом:

Через промт: Генерация по описанию.

Промт + Шаблон: Выбор структуры + описание.

По фото (Vision): Загрузка изображения-примера (используй возможности Gemini 3.1 Pro по анализу изображений).

Редактор: Экран, разделенный на две части: слева — чат с ИИ для правок, справа — интерактивный Live-preview сайта.

Экспорт: Кнопка «Скачать ZIP» и кнопка «Опубликовать на хостинг» (реализовать как заглушку с реферальной ссылкой).

Логика генерации (Prompt Engineering):

Любой запрос пользователя автоматически дополняй системными требованиями: «Используй современные теги HTML5, семантическую верстку для SEO, добавь мета-теги, пропиши Alt для изображений, используй сложные тени для скевоморфного стиля и интерактивные SVG-анимации».

Код должен быть адаптивным (Mobile First).

Монетизация:

Тарифная сетка: Реализовать логику проверки лимитов (кредитов/токенов) перед генерацией.

Реферальная программа: В разделе экспорта добавить баннеры рекомендуемых хостингов.

Интеграции:

Gemini 3.1 Pro API.

Kie api для генерации картинок в Nano Banana и видео Seedance 2.0. так же через экран редактора. далее нужно вставлять это фото и видео в нужный раздел сайта через промт юзера. На выходе пользователь скачивает ZIP архив с фото и видео файлами а так же HTML код сайта.
